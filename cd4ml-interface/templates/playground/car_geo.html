Documentação da API

Endpoint: POST /geografico/car_geo

Parâmetros

  • codigo_ibge: string - Código IBGE do município

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/geografico/car_geo",
    data={"codigo_ibge": "3550308"},
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
