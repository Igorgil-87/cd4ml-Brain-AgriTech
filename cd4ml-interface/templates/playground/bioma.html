Documentação da API

Endpoint: POST /socioambiental/bioma

Parâmetros

  • latitude: float - Latitude da localização
  • longitude: float - Longitude da localização

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/socioambiental/bioma",
    data={
        "latitude": -15.7941,
        "longitude": -47.8825
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
