Documentação da API

Endpoint: POST /logistica/distancia-rios

Headers: X-API-Key

Parâmetros

  • latitude: float
  • longitude: float

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/logistica/distancia-rios",
    data={"latitude": -21.12345, "longitude": -47.6789},
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)

print(response.json())
