Documentação da API

Endpoint: POST /geodesica

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • municipio: string - Nome do município
  • estado: string - Sigla do estado (UF)
  • descricao_imovel: string - Descrição da propriedade

API Key

Sua chave: {{ api_key or 'SUA_CHAVE_AQUI' }}

Exemplo

import requests

requests.post(
    "http://localhost:11001/geodesica",
    data={
        "municipio": "Ribeirão Preto",
        "estado": "SP",
        "descricao_imovel": "Fazenda Modelo"
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
