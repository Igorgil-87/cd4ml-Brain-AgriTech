Documentação da API

Endpoint: POST /produtividade/safrahistorica

Parâmetros

  • cultura: str - Nome da cultura (ex: soja, milho)
  • estado: str - Sigla do estado (ex: SP, MT)
  • ano: int - Ano da safra (ex: 2023)

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/produtividade/safrahistorica",
    data={
        "cultura": "soja",
        "estado": "MT",
        "ano": 2023
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
