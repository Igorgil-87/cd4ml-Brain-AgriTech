Documentação da API

Endpoint: POST /score_gove

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • documento: string - CPF ou CNPJ do produtor
  • area_total: float - Área total da propriedade
  • qtd_embargos: int - Quantidade de embargos registrados
