Documentação da API

Endpoint: POST /socioambiental/unidade_conservacao_socio

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • latitude: float - Latitude da área
  • longitude: float - Longitude da área

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/socioambiental/unidade_conservacao_socio",
    data={
        "latitude": -15.4321,
        "longitude": -47.6543
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
