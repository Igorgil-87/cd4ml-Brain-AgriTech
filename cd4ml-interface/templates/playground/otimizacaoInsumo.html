Documentação da API

Endpoint: POST /otimizacaoInsumo

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • area_plantada: float - Área total da lavoura em hectares
  • tipo_insumo: string - Nome do insumo agrícola
  • qtd_insumo: float - Quantidade disponível em kg

API Key

Sua chave: {{ api_key or 'SUA_CHAVE_AQUI' }}

Exemplo

import requests

requests.post(
    "http://localhost:11001/otimizacaoInsumo",
    data={
        "area_plantada": 100,
        "tipo_insumo": "Fertilizante A",
        "qtd_insumo": 500
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
