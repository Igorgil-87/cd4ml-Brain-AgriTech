Documentação da API - Bioma

Endpoint: POST /socioambiental/bioma

Método: POST

Headers: X-API-Key

Parâmetros

  • latitude: float - Ex: -10.123456
  • longitude: float - Ex: -55.654321

API Key

Chave de autenticação: {{ api_key or 'SUA_CHAVE_AQUI' }}

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/socioambiental/bioma",
    data={
        "latitude": -10.123456,
        "longitude": -55.654321
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)

print(response.json())
