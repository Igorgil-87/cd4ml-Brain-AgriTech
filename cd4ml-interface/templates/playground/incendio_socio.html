Documentação da API

Endpoint: POST /socioambiental/incendio_socio

Parâmetros

  • latitude: float - Latitude da área
  • longitude: float - Longitude da área

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/socioambiental/incendio_socio",
    data={
        "latitude": -10.1234,
        "longitude": -48.5678
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
