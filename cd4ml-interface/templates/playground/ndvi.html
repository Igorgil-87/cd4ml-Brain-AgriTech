Documentação da API

Endpoint: POST /produtividade/ndvi

Parâmetros

  • valor_infravermelho: float - Valor da banda NIR da imagem
  • valor_vermelho: float - Valor da banda vermelha

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/produtividade/ndvi",
    data={
        "valor_infravermelho": 0.7,
        "valor_vermelho": 0.3
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
