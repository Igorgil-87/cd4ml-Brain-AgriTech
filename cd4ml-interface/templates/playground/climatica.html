📚 Documentação da API

Endpoint: POST /climatica

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • temperatura_media – Temperatura média esperada
  • precipitacao – Volume de chuva em milímetros
  • umidade – Umidade relativa do ar em porcentagem

API Key

Utilize sua chave de acesso: {{ api_key or 'SUA_CHAVE_AQUI' }}

Exemplo de Chamada (Python)

import requests

response = requests.post(
    "http://localhost:11001/climatica",
    data={
        "temperatura_media": 26.4,
        "precipitacao": 12.3,
        "umidade": 75
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
