Documentação da API

Endpoint: POST /socioambiental/indigina_socio

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • latitude: float - Latitude do ponto consultado
  • longitude: float - Longitude do ponto consultado

Exemplo (Python)

import requests

response = requests.post(
    "http://localhost:11001/socioambiental/indigina_socio",
    data={
        "latitude": -10.1234,
        "longitude": -48.5678
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
print(response.json())
