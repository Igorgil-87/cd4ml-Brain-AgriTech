Documentação da API

Endpoint: POST /colheita

Content-Type: application/x-www-form-urlencoded

Parâmetros

  • lat_inicial: float - Latitude do ponto inicial
  • lon_inicial: float - Longitude do ponto inicial
  • distancia_km: float - Distância desejada em quilômetros

Exemplo

import requests

requests.post(
    "http://localhost:11001/colheita",
    data={
        "lat_inicial": -21.123,
        "lon_inicial": -48.321,
        "distancia_km": 5.0
    },
    headers={"X-API-Key": "SUA_CHAVE_AQUI"}
)
